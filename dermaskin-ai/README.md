# DermaSkin AI — 皮肤科技能操作训练分析评估软件

> 面向皮肤科住院医师规范化培训的技能操作训练分析评估系统

## 快速启动

### 环境要求
- Node.js >= 18
- npm >= 9
- Conda (可选，用于 Python AI 处理组件)

### 安装与运行

```bash
# 进入项目目录
cd dermaskin-ai

# 安装依赖
npm install

# ═══ Web 开发模式 ═══
npm run dev

# ═══ Electron 桌面开发模式 ═══
npm run dev:electron

# ═══ 构建生产版本 ═══
npm run build:all      # 构建 Web + Electron

# ═══ 打包桌面安装程序 ═══
npm run dist           # 生成 Windows 安装包 (release/)
```

### Conda 环境（AI 处理组件）

```bash
conda activate dermaskin
```

## 功能模块

### 1. 操作技能评估中心
- **班级概览**：玻璃拟态 KPI 卡片（悬停动效）、综合分析报告
- **高级图表**：南丁格尔玫瑰图（阴影深度）、旭日图（层级钻取）、堆叠面积图（渐变填充）、知识树图（课程-技能-EPA层级）
- **学员散点图**：注意力-成绩分布（焦点聚焦效果）
- **维度热力图**：学员×维度成绩矩阵（6级绿色渐变）
- **数据来源标识**：AI 分析 / 训练数据 来源徽章
- **AI 智能洞察**：自动分析班级强弱维度、风险学员、AI 分析覆盖率
- **学员详情弹窗**：视频回放、评分表、循证证据卡片、雷达图对比、AI建议、趋势分析、学习行为漏斗、跨课程对比

### 2. 循证教学中心
- **能力框架**：EPA 体系展示、里程碑等级、达成度雷达图
- **教学分析**：Sankey 知识流图（教学模块→知识域→技能维度→EPA）、交互式旭日图、力导向概念网络图
- **循证评估**：极坐标堆叠图（Kirkpatrick四级）、矩形树图（Bloom认知层次）、动态仪表盘KPI
- **成效追踪**：学员成长曲线、EPA 里程碑进度
- **决策支持**：风险预警散点图、PDCA 循环进度卡片

### 3. 视频分析工作台（真实AI分析）
- **视频导入**：拖拽上传，支持 MP4/MOV/AVI 格式
- **Qwen-VL 多模态分析**：
  - 自动提取视频关键帧（可配置间隔和最大帧数）
  - 发送帧到 Qwen-VL-Max 进行视觉理解和评分
  - 基于课程评分量表的结构化提示词工程
  - 返回每维度得分、时间戳、循证依据
- **Qwen-Audio 语音转录**：
  - 自动提取视频音频轨道
  - 语音转文字（说话人分段：教师/学员）
  - 沟通能力评分与分析
  - 专业术语识别
- **6步骤分析流水线**：帧提取 → 音频提取 → AI视觉分析 → 语音转录 → 语音分析 → 生成报告
- **详细报告视图**：
  - **真实视频帧截图** + AI标注时间戳
  - 每维度循证评分依据（AI生成的证据文本 + 截图描述 + 改进建议）
  - 语音转录展示（说话人分段对话格式、沟通评分、专业术语标签）
  - 关键操作节点时间线（good/neutral/concern标注）
  - AI 整体评价与循证教学建议
  - 评分雷达对比（AI评分 vs 班级均值）
  - 维度得分趋势分析图

### 4. 学员管理
- 搜索与分组筛选
- 增强表格（技能进度可视化、成绩趋势迷你图、等级徽章）
- 详情面板（EPA雷达图、多课程对比、成长趋势、EPA里程碑）
- KPI 统计（总人数、均分、优秀人数、需关注人数）

### 5. AI 教学助手
- 现代对话界面（渐变气泡、动画头像）
- **真实 AI 对话**：集成通义千问 qwen-turbo-latest，基于系统数据的智能回复
- **页面上下文感知**：根据当前浏览页面动态调整系统提示词和快捷问题
- **AI/离线双模式**：有 API Key 时使用真实 AI，无 Key 时自动降级为规则引擎
- 快捷提问模板（随页面切换动态变化）
- 打字指示器动画

### 5.5 AI 悬浮助教球（全局）
- **全页面可用**：固定在右下角，所有页面均可访问
- **实时数据汇总**：跨课程班级统计、薄弱学员识别、分析覆盖率
- **页面智能洞察卡片**：评估中心/循证教学各 Tab/决策支持均内嵌 AI 分析

### 6. 报告中心
- **班级报告**：组合柱状+折线图、维度雷达图、排名表（前三高亮）、跨课程对比
- **个人报告**：学员信息卡片、维度评分分析、证据文本、AI建议
- **AI 分析洞察整合**：自动展示真实 AI 视频分析结果（如有）
- **导出功能**：PDF（Electron 原生 printToPDF）、Excel（CSV）、直接打印

### 7. 系统设置
- **AI 配置**：API密钥、基础地址、模型选择、Token/温度参数、并发数
- **界面设置**：主题色（5种）、字体大小、侧边栏位置、卡片圆角、动画开关、紧凑模式
- **通知与提醒**：分析完成通知、风险预警、预警阈值、自动保存间隔、声音提醒
- **数据管理**：导出数据、备份/恢复、缓存管理、重置设置

### 8. 自动更新
- 基于 electron-updater 的自动更新机制
- 启动后 10 秒自动检查更新
- 非侵入式更新通知栏（下载/安装/稍后提醒）
- 下载进度实时显示
- 一键重启安装

### 9. 数据联动架构
- **统一数据 Hook**（useUnifiedData）：合并静态训练数据 + 实时 AI 分析结果
- **跨页面数据流**：视频分析结果自动回流到评估中心、报告中心、学员详情
- **数据来源标识**：每条数据标注来源（AI 分析 / 训练记录）
- **SQLite 持久化**：分析历史、学员数据、评估记录本地存储

## AI 多模态集成

### 配置步骤
1. 在 [阿里云 DashScope](https://dashscope.console.aliyun.com/) 注册并获取 API Key
2. 在系统设置 → AI 配置 中填入 API Key
3. 选择视觉模型（推荐 qwen-vl-max-latest）和语音模型
4. 上传操作视频，点击「开始 AI 分析」

### 支持的模型
| 模型 | 用途 | 说明 |
|------|------|------|
| qwen-vl-max-latest | 视觉分析 | 最强视觉理解能力（推荐） |
| qwen-vl-plus-latest | 视觉分析 | 均衡性能 |
| qwen-omni-turbo-latest | 视觉分析 | 快速响应 |
| qwen-audio-turbo-latest | 语音识别 | 通用语音转录（推荐） |
| gpt-4o | 视觉分析 | OpenAI 兼容 |

### 分析流程
```
视频文件 → 帧提取(Canvas) → Qwen-VL评估 → 结构化评分
         → 音频提取(WebAudio) → Qwen-Audio转录 → 语音分析
         → 组合报告（循证依据 + 语音转录 + 教学建议）
```

详见 [AI集成架构文档](docs/06-AI集成架构.md)

## 课程配置

系统支持 5 门皮肤科操作课程：
1. **真菌镜检操作** - 5个评分维度，满分100
2. **皮肤活检术** - 6个评分维度，满分100
3. **斑贴试验** - 5个评分维度，满分100
4. **Tzanck涂片检查** - 5个评分维度，满分100
5. **皮肤镜检查** - 5个评分维度，满分100
