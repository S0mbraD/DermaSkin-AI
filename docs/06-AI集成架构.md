# AI 多模态集成架构

## 概述

DermaSkin AI 采用通义千问（Qwen）多模态大模型进行视频分析，实现真正可用的操作技能评估系统。

## 技术架构

```
视频文件 (MP4/MOV/AVI)
    │
    ├── 视频帧提取 (Canvas API)
    │   └── N 帧关键帧 (JPEG base64, 768px宽)
    │
    ├── 音频提取 (Web Audio API)
    │   └── WAV 16kHz 单声道
    │
    ├── Qwen-VL 视觉分析
    │   ├── 系统提示词 (课程评分量表)
    │   ├── 用户消息 (帧图片 + 评估指令)
    │   └── 返回: 结构化JSON评估结果
    │
    └── Qwen-Audio 语音转录
        ├── 音频数据 (base64 WAV)
        └── 返回: 转录文本 + 说话人分段
```

## 模型配置

### 视觉分析模型
| 模型 | 特点 | 推荐场景 |
|------|------|---------|
| qwen-vl-max-latest | 最强视觉理解 | 精确评估（推荐） |
| qwen-vl-plus-latest | 均衡性能 | 日常使用 |
| qwen-omni-turbo-latest | 快速响应 | 批量分析 |

### 语音识别模型
| 模型 | 特点 |
|------|------|
| qwen-audio-turbo-latest | 通用语音识别（推荐） |
| qwen2-audio-instruct | 指令跟随型 |

## 提示词工程

### 评估系统提示词结构
1. **角色定义**: 资深皮肤科临床教学专家
2. **课程信息**: 课程名称 + 评分维度及满分
3. **评分标准**: 优秀(≥85%) / 合格(60-84%) / 不合格(<60%)
4. **输出格式**: 严格JSON，包含:
   - `scores[]`: 每维度得分
   - `timestamps[]`: 关键操作时间戳
   - `evidenceDetails[]`: 循证评分依据（含截图描述、证据文本、改进建议）
   - `overallComment`: 整体评价
   - `teachingSuggestions`: 循证教学建议
   - `keyMoments[]`: 关键操作节点时间线

### 语音分析提示词
- 说话人分段（教师/学员）
- 关键指导提取
- 沟通能力评分
- 专业术语识别

## 分析流程

1. **帧提取** (5%→20%): 按间隔提取视频关键帧，缩放至768px
2. **音频提取** (20%→25%): 提取音频轨道，转换为16kHz WAV
3. **AI视觉分析** (30%→65%): 发送帧到Qwen-VL，获取结构化评估
4. **语音转录** (70%→80%): 发送音频到Qwen-Audio
5. **语音分析** (85%→95%): 分析转录内容（沟通能力、术语使用）
6. **生成报告** (95%→100%): 组合所有结果

## 报告内容

### 视觉分析报告
- 每维度评分 + 循证依据
- 视频帧截图 + 时间戳
- 关键操作节点时间线
- 整体评价与教学建议

### 语音转录报告
- 说话人分段（教师/学员交替）
- 沟通能力评分
- 专业术语识别
- 教学互动分析

## AI 全面整合

### AI 悬浮助教球
- 使用 **qwen-turbo-latest** 模型进行实时对话
- 系统提示词注入当前页面上下文 + 数据摘要
- 有 API Key 时为真实 AI 模式，无 Key 时自动降级为规则引擎
- 快捷问题随页面动态变化

### AI 智能洞察卡片
各页面内嵌 AI 分析洞察（AIInsightCard 组件）：

| 页面 | 洞察内容 |
|------|---------|
| 评估中心 | 班级强弱维度、风险学员、AI 分析覆盖率 |
| 教学分析 | 知识流路径、教学方法优化 |
| 循证评估 | Kirkpatrick/Bloom/EPA 达成度 |
| 成效追踪 | 成长轨迹、EPA 里程碑进度 |
| 决策支持 | 风险预测、PDCA 干预建议 |
| 能力框架 | EPA 覆盖分布、能力达标分析 |
