# 架构设计文档

## 1. 模块架构

```
DermaSkin AI
├── 操作技能评估中心（主页）
│   ├── 班级概览面板 (OverviewTab)
│   ├── 分析报告 (ReportTab)
│   └── 学员详情弹窗 (StudentDetailModal)
│
├── 循证教学中心
│   ├── 能力框架 (FrameworkTab)
│   ├── 教学分析 (TeachingAnalysis)
│   ├── 循证评估 (EvidenceQualityAnalysis)
│   ├── 成效追踪 (LearningOutcomeTracking)
│   └── 决策支持 (EvidenceDecisionSupport)
│
├── 视频分析工作台 (VideoWorkbench)
├── 学员管理 (StudentManagement)
├── AI 助手 (AIAssistant)
└── 报告中心 (ReportCenter)
```

## 2. 目录结构

```
dermaskin-ai/
├── src/
│   ├── main.tsx                 # 入口文件
│   ├── App.tsx                  # 根组件（路由配置）
│   ├── index.css                # 全局样式 + CSS Variables
│   ├── types/                   # TypeScript 类型定义
│   ├── data/                    # 课程配置 + 模拟数据
│   ├── stores/                  # Zustand 全局状态
│   ├── utils/                   # 工具函数
│   ├── components/              # 通用组件
│   │   ├── Layout/              # 布局组件
│   │   └── common/              # 公共 UI 组件
│   └── pages/                   # 页面组件
│       ├── EvalCenter/          # 评估中心
│       ├── EvidenceBased/       # 循证教学
│       ├── VideoWorkbench/      # 视频工作台
│       └── ...
├── electron/                    # Electron 主进程
├── public/                      # 静态资源
├── docs/                        # 开发文档
└── package.json
```

## 3. 数据流

```
外部输入 → Zustand Store → React 组件（自动重渲染）
                ↑                    ↓
           SQLite（持久化）     用户交互（修改评分等）
```

## 4. 状态管理策略

| Store | 数据 | 职责 |
|-------|------|------|
| useStudentStore | students[] | 学员 CRUD + EPA 进展 |
| useCourseStore | courses[], activeCourseId | 课程管理 + 评分更新 |
| useSessionStore | sessions[] | 课堂分析会话管理 |
| useEvidenceStore | evidence[] | 证据档案管理 |
| useAnalysisStore | kirkpatrick, bloom, risk | 分析数据计算 |
