# DermaSkin AI — 项目总览

## 1. 项目定位

DermaSkin AI 是一套面向皮肤科住院医师规范化培训的技能操作训练分析评估软件。

## 2. 核心工作流

```
皮肤科技能操作视频（录制或导入）
         ↓
AI 多模态分析（视觉识别 + 语音转写）
         ↓
逐维度自动评分 + 视频时间戳定位
         ↓
生成循证评分依据文本（三级：优秀 / 合格 / 不合格）
         ↓
写入学员档案 → 班级分析 → EPA 能力追踪 → 教学决策
```

## 3. 目标用户

| 角色 | 主要功能 |
|------|---------|
| 皮肤科教学教师 | 评分、查看报告、下达教学建议 |
| 皮肤科住院医师 | 查看个人档案、视频复盘、接受反馈 |
| 培训管理人员 | 班级统计、EPA 达成率、课程质量分析 |

## 4. 技术栈

| 层次 | 技术 | 版本 |
|------|------|------|
| 前端框架 | React + TypeScript | React 18 |
| 构建工具 | Vite | v5 |
| UI 组件库 | Ant Design | v5 |
| 数据可视化 | Apache ECharts | echarts-for-react |
| 动画 | Framer Motion | latest |
| 状态管理 | Zustand | v4+ |
| 路由 | React Router | v6 |
| 桌面封装 | Electron | v28+ |

## 5. 设计哲学

1. **循证优先**：任何评分结论必须关联到视频中可定位的时间戳和操作片段
2. **数据驱动**：所有文本建议、等级判断均由结构化数据计算得出
3. **可扩展课程**：新增课程只需补充 CourseConfig 对象
4. **离线可用**：Electron 封装后可在无网络环境完整运行
5. **专业美观**：医疗级别的严肃专业感，浅色主题风格
