# UI 设计规范

## 1. 设计原则

- **浅色主题**：专业医疗教学软件，以白色/浅灰为主基调
- **信息层次清晰**：通过字号、颜色、间距区分信息优先级
- **图表丰富直观**：大量使用 ECharts 数据可视化
- **微动效提升体验**：Framer Motion 卡片悬浮、列表入场动画

## 2. CSS Variables

```css
:root {
  --primary:        #4361EE;   /* 蓝紫，主交互色 */
  --success:        #05CD99;   /* 翠绿，优秀/成功 */
  --warning:        #F59E0B;   /* 琥珀，警示/中等 */
  --danger:         #EF4444;   /* 红，危险/不合格 */
  --purple:         #7C3AED;   /* 辅助紫 */
  --bg-main:        #F4F7FE;   /* 页面背景 */
  --bg-card:        #FFFFFF;   /* 卡片背景 */
  --text-heading:   #2B3674;   /* 标题 */
  --text-secondary: #707EAE;   /* 正文 */
  --text-muted:     #A3AED0;   /* 辅助说明 */
  --border-main:    #E9EDF7;   /* 边框 */
}
```

## 3. 组件视觉规范

| 元素 | 值 |
|------|---|
| 大卡片圆角 | 14px |
| 中卡片圆角 | 12px |
| 小内嵌块圆角 | 10px |
| 卡片阴影 | 0 1px 3px rgba(0,0,0,0.04) |
| Hover 阴影 | 0 6px 20px rgba(0,0,0,0.08) |
| KPI 数字 | 18-28px, weight 800 |
| 卡片标题 | 13px, weight 700 |
| 正文 | 12-13px |
| 辅助说明 | 10-11px |

## 4. 等级颜色系统

| 等级 | 颜色 | 标签 | 判定标准 |
|------|------|------|---------|
| 优秀 | #05CD99 | 优秀 | 得分率≥85% |
| 合格 | #4361EE | 合格 | 得分率60-84% |
| 不合格 | #EF4444 | 不合格 | 得分率<60% |

综合评分：≥90优秀(绿) ≥80良好(蓝) ≥70合格(琥珀) <70需加强(红)
